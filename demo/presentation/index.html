YAHS

Yet Another HTML5 Slideshow

Features

Why another HTML5 slideshow ?

Because this one is :

  • Really easy to code with simple tags
  • Really easy to customize with specific styles
  • Browsable on tablets and phones
  • Printable in landscape and portrait orientation
  • Distribuable as a single HTML page

Slide number

Simply add a div with 'slidenumber' class to display the slide number in footer

Usage

  • Pre-requisite :
    • Install node.js
    • Install grunt
    • Install dependencies with npm : 
      > cd /workdir/yahs
> npm install
  • Test your local slideshow in dev mode :
    • Disable your browser security : because of local access (url file://)
    • Open the local page in your browser
  • Build the dist page with grunt : 
    > grunt
  • Browse the all-in-one dist page

Slide templates

  • Check the samples (under /demo/) :
    • presentation : this slides (basic template)
    • nodejs-presentation : real presentation made for a french seminar the 02/28/2013
  • Start with the basic template
  • Hack it!

Advices

When grunt build is done, images, custom fonts, and scripts are all compiled/minified/inlined in the dist page.

To have a ready-to-use single page, you have to be carefull with links potentially broken.
